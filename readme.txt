Инструкция для сайта блюд

Распаковываем файлы, так чтобы папка application и другие были в корне!
К примеру в denwer я распаковал сюда Z:\home\courses.ru\www.

*** НАСТРОЙКА ПОДКЛЮЧЕНИЯ К БД ***
Нужно создать БД и залить туда исходники.
Исходники базы данных находятся здесь application\data\mysql.sql.
Потом нужно настроить подключение, открываем application\config\main.php 
и меняем под себя, использовалось PDO, поэтому сайт может работать с любой СУБД.
        'dbConnect' => 'mysql:host=localhost;dbname=courses',
        'dbUser' => 'root',
        'dbPassword' => '',


*** НАСТРОЙКА САЙТА ***
Параметры приложения относящие к самому сайту можно найти тут application\config\main.php
Количество блюд которых может выбрать пользователь из какжой группы
	'coursePerGroup' => 1 

Учетная запись админа
	'adminLogin' => 'admin',
        'adminPassword' => '12345'

Также сайт может поддерживать еще другие группы блюд помимо гарнир и основное,
для этого нужно просто заполнить таблицу в БД category.

*** АДМИНИСТРИРОВАНИЕ ***
Для захода в админку нужно вбить URL/site/admin.
Там 3 вкладки
1) Блюда - манипулирование ими
2) Заказы - нужно выбрать пользователя, у котого хотим увидеть заказ, и кликаем,
после чего будет отображен его заказ
3) За период - Можно получить информацию о заказаз за определенный период
